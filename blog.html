Important FAQ

What is the Document Object Model (DOM)?

The DOM is the bridge between web pages (HTML/CSS) and programming languages like JavaScript. It allows you to access and manipulate the content, structure, and styles of a webpage dynamically. The browser turns your HTML code into a structured representation called the DOM, which you can modify using JavaScript to create interactive and dynamic websites.

How do you select an element from the DOM?

I can select an element from the DOM on multiple way.
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

What is event delegation in the context of the DOM and why is it useful?

When you want to do something on many items in your web page, you usually add a "listener" to each item. But if you have a lot of items, adding a listener to each one can slow things down. Event delegation is a clever way to solve this problem by adding just one listener to a "parent" element instead of to each "child" item.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles in the DOM like this:
Attributes:
- setAttribute('attr', 'value'): Change or add
- an attribute.
- getAttribute('attr'): Get an attribute's value.
- removeAttribute('attr'): Remove an attribute.

Styles:
- element.style.property = 'value': Change a specific CSS property.

Classes:
- element.classList.add('className'): Add a class.
- element.classList.remove('className'): Remove a class.
- element.classList.toggle('className'): Toggle a class on/off.

Content:
- element.innerHTML = 'HTML content': Change HTML inside theelement.
- element.textContent = 'Text content': Change only text content.
